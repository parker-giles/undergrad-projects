# undergrad-projects
A small collection of statistics and mathematics projects/exercises I completed during my undergraduate tenure.  
The "undergraduate_projects" folder contains several of these.
"coupon_collector_bask_calc_new" is a calculator in Python for the numerical and approximate E(X) and Var(X) of the statistics problem: The Coupon Collector's Problem.
"non_pivot_guass_elem" is an algorithm in Python that performs Guassian elemination and back subsitution for a square matrix.
"pivot_guass_elem" is the same as "non_pivot_guass_elim" but implements row swaps.
"nonpivot_lu_factorization" is an algorithm in Python that performs LU factorization on a matrix.
"pivot_lu_factorization" works similarly to "nonpivot_lu_factorization" but implements row swaps and logs them in a permutation matrix. This one is closer to a recreation of the LU factorization function in Numpy.
